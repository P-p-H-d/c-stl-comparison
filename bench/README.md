# Run

Just run `make` to run the benchmark after having running the toplevel Make to get the external dependencies.

# Environment variable

* `QT`   : Path of the directory containing the pkgconfig file of QT5
* `GLIB` : Path of the directory containing the pkgconfig file of GLIB2
* `SPARSEPP`: Path to the sparsepp headers
* `DENSEHASHMAP`: Path to the google dense hashmap headers
*
